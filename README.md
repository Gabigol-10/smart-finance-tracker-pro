# 💰 Smart Finance Tracker Pro

Aplicación web moderna para gestión de finanzas personales con interfaz intuitiva y análisis en tiempo real.

## ✨ Características

- 📊 Dashboard financiero con gráficos interactivos
- 💵 Gestión de ingresos y gastos por categorías
- 🎯 Metas de ahorro con seguimiento de progreso
- 📅 Calendario financiero
- 📈 Panel de análisis y reportes
- 🌗 Modo claro/oscuro
- 📱 Diseño responsive

## 🚀 Tecnologías

- React 19
- React Router DOM
- Chart.js & Recharts
- Bootstrap 5
- Framer Motion
- SweetAlert2
- Vite

## 📦 Instalación

```bash
# Clonar repositorio
git clone https://github.com/Gabigol-10/smart-finance-tracker-pro.git

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build
```

## 🎯 Uso

1. La aplicación inicia en el Dashboard principal
2. Usa el sidebar para navegar entre secciones
3. Registra tus ingresos y gastos
4. Crea metas de ahorro
5. Visualiza análisis y reportes
6. Exporta datos en PDF o Excel
